# YouTube Explorer

A React application for exploring and discovering YouTube content with an intuitive user interface.

## Features

- Search YouTube videos by keywords
- View video details including title, description, and thumbnail
- Watch video previews
- Responsive design for all devices
- Clean and modern UI

## Installation

1. Clone the repository:

```bash
git clone https://github.com/abdaurhman-mabrouk/youtube-explorer.git
cd youtube-explorer
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file and add your YouTube API key:

```
REACT_APP_YOUTUBE_API_KEY=your_api_key_here
```

## Usage

Start the development server:

```bash
npm start
```

The application will open at `http://localhost:5173`

## Build

Create a production build:

```bash
npm run build
```

## Technologies Used

- React
- React Hooks
- YouTube Data API v3
- CSS/Responsive Design
